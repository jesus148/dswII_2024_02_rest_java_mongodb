Microservcio :
Un microservicio REST es una pequeña aplicación independiente que cumple una función específica dentro de una arquitectura más grande. Se comunica con otros microservicios y clientes a través de interfaces RESTful, utilizando HTTP para las solicitudes y respuestas. Aquí tienes una descripción básica y algunos pasos para crear un microservicio REST:
